# Saldo – Payment Simulation App

⚠️ **Disclaimer**

Saldo is a **virtual payment simulation app** created for educational and demonstration purposes only.

- No real money is used
- No banking or UPI networks are involved
- All transactions are simulated
- This app should NOT be used for real financial activity

---

## About the Project

Saldo demonstrates how modern QR-based payment apps work at a UI/UX and logic level.

The app allows users to:
- Register and login locally
- Set and verify a secure PIN
- Scan any QR code
- Enter a payment amount
- Complete a simulated payment
- Hear a confirmation sound
- View updated virtual balance

---

## Features

- QR code scanning (pay anyone)
- PIN-based verification
- Virtual editable balance
- Transaction success feedback with sound
- Local data persistence
- Flutter Web support

---

## Tech Stack

- Flutter (Web)
- Dart
- Local storage (SharedPreferences)
- Netlify (Deployment)
